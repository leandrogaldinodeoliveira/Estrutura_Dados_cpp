# Estruturas de Dados em C++

Este repositório reúne implementações didáticas de estruturas de dados em C++, desenvolvidas como parte dos meus estudos em Programação Orientada a Objetos e Algoritmos.

## Estruturas implementadas

- Pilha
- Fila
- Lista

## Objetivo

O objetivo deste repositório é estudar a implementação de estruturas de dados fundamentais em C++, explorando conceitos como:

- alocação dinâmica de memória;
- classes e objetos;
- encapsulamento;
- construtores e destrutores;
- ponteiros;
- operações de inserção, remoção, busca e exibição.

## Organização

Cada estrutura possui sua própria pasta, contendo os arquivos de implementação e um programa principal para testes.
